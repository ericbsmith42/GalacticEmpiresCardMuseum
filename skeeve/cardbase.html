Cards by Function

On occasion I have need to know or have been asked about what cards can be used to fulfill a certain function. This page is an attempt to keep that question answered. Below are several categories of card functions and card rules which seem to come up again and again. It would have been really cool to make this an on-line queriable database, but my web space provider don't like custom programming because of the potential security hole. Them's the breaks.

Just in case there is a particular function you want to see, here's a list of all of the current categories. Feel free to suggest any others.

  • Control Opponent's Ship
  • Extra Card Draws
  • Extra Card Plays
  • Force Opponent to Discard from Hand
  • Remove from Game
  • Remove or Discard from Play
  • Resource Hosers
  • Retrieve Card from Discard Pile
  • Use Foreign or Exclusive Technology
  • View or Modify Top of Deck
  • View Opponent's Hand or Draws (Scout Functions)
These lists are current as of the release of Galactic Invaders. If you know of any card up to that point which should be added to any category let me know.
